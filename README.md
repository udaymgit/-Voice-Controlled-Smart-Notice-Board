# 🎙️ Voice Controlled Smart Notice Board using ESP32

## 📌 Overview

The **Voice Controlled Smart Notice Board** is an IoT-based project developed using an **ESP32 NodeMCU**, **16×2 LCD**, Bluetooth, and Wi-Fi. The system allows users to send notice messages through voice commands using an Android smartphone. The spoken message is converted into text by the phone and transmitted to the ESP32 via Bluetooth, where it is displayed on the LCD.

This project was originally developed as a **college mini project** and has been recreated and documented for learning and portfolio purposes.

---

## 🚀 Features

* Voice-controlled notice display
* ESP32 NodeMCU based implementation
* Bluetooth communication
* Wi-Fi ready architecture for future expansion
* 16×2 LCD display
* Automatic scrolling for long messages
* Low-cost IoT solution

## ⚙️ Working

1. User speaks a notice into an Android phone.
2. The phone converts speech into text.
3. The text is transmitted to the ESP32 through Bluetooth.
4. ESP32 receives the message.
5. The notice is displayed on the 16×2 LCD.
6. Long messages scroll automatically across the display.
